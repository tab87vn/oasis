# Oasis

Oasis is a simple Data Lake written in Ruby and Hanami

[![Build Status](https://travis-ci.org/toch/oasis.svg?branch=master)](https://travis-ci.org/toch/oasis)

## Usage

```Bash
bundle
hanami db prepare
hanami server
```

## Run the tests

```Bash
bundle
HANAMI_ENV=test hanami db prepare
rake
```
